client.view

function client.view(register_key: string) -> Table

Requests and caches a registered table from the server.

Returns: The replicated table

client.view_as_proxy

function client.view_as_proxy(register_key: string)
    -> RegistreeInterface<any, (RemoteEvent?)>

Gets a proxy interface for the table.

Example:

local proxy = ReplicatedRegistry.client.view_as_proxy("player_data")
proxy.set("coins")(proxy.get("coins") + 10)
proxy.replicate() -- Send to server
